# Java 3D Engine

## Controls;
- **Movement**
 - **WASD** - **Walk**
 - **Arrow keys (left&right)** - **Turn camera**
 - **Q** - **Dash**
 - **Space** - **Jump**
 
* **Misc**
 * **F1** - **Toggle Debug mode (Might affect performance on lower-end devices.)**
 * **Z** - **Toggle wireframe rendering**
 * **X** - **Toggle transparent rendering (LAGGY)**
